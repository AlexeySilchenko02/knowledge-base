1. Выполняем следующую команду:

```bash

manage-bde -off C:

```

При вот таком выводе:

```bash

PS C:\Users\Al200> manage-bde -off C:

Шифрование дисков BitLocker: версия средства настройки:

(C) Корпорация Майкрософт (Microsoft Corporation), 2013. Все права защищены.

  

ОШИБКА: в этом томе хранятся внешние ключи, которые могут автоматически снимать

блокировку c других томов. Прежде чем расшифровывать том, необходимо удалить

эти ключи.

  

НЕОБХОДИМЫЕ ДЕЙСТВИЯ:

  

2. Для предотвращения потери данных убедитесь, что для соответствующих томов

   данных существует пароль восстановления или ключ восстановления.

  

3. Введите команду "manage-bde -autounlock -ClearAllKeys <Том>", чтобы удалить

   сохраненные внешние ключи

```

4. Выполняем следующую команду:

```bash

manage-bde -autounlock -clearallkeys C:

```

5. Если есть другие диски:

```bash

manage-bde -off D:

```

6. Следим за процессов расшифровки:

```bash

manage-bde -status

```
